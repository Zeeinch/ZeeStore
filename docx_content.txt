Kata Pengantar
Puji syukur kehadirat Tuhan Yang Maha Esa atas segala rahmat dan karunia-Nya, sehingga penulis dapat menyelesaikan laporan tugas yang berjudul "Website ZeeStore - Platform Marketplace Online UMKM" ini dengan baik dan tepat waktu.

Penulis menyadari bahwa laporan ini masih jauh dari kesempurnaan. Oleh karena itu, kritik dan saran yang membangun sangat penulis harapkan demi perbaikan di masa mendatang. Akhir kata, semoga laporan ini dapat bermanfaat bagi pembaca.

Bab I Pendahuluan
1.1 Latar Belakang
Di era digital saat ini, e-commerce atau perdagangan elektronik telah menjadi salah satu pilar utama perekonomian modern. Pertumbuhan pengguna internet di Indonesia yang mencapai lebih dari 200 juta orang membuka peluang besar bagi pelaku Usaha Mikro, Kecil, dan Menengah (UMKM) untuk memasarkan produk mereka secara online. Platform marketplace seperti Tokopedia dan Shopee telah membuktikan bahwa model bisnis ini sangat efektif dalam menjangkau konsumen luas.
Berdasarkan latar belakang tersebut, kami mengembangkan ZeeStore, sebuah website marketplace yang dirancang untuk memfasilitasi jual beli produk UMKM secara online. Website ini dibangun menggunakan HTML5, CSS3 (Bootstrap 5), dan JavaScript vanilla dengan backend Supabase (PostgreSQL + Authentication). Website terdiri dari 7 halaman HTML yang saling terintegrasi dan telah dilakukan berbagai modifikasi kreatif pada warna, tampilan, layout, dan fungsionalitas.
1.2 Rumusan Masalah
Berdasarkan latar belakang di atas, rumusan masalah pada laporan ini adalah:
1. Bagaimana membangun website profil usaha/bisnis UMKM berupa marketplace online yang fungsional?
2. Bagaimana menerapkan modifikasi kreatif pada warna, tampilan, posisi teks, layout, dan animasi secara manual maupun dengan bantuan AI?
3. Bagaimana mengintegrasikan website dengan backend database dan autentikasi pengguna?
1.3 Manfaat dan Tujuan
Tujuan:
1. Memenuhi tugas mata kuliah Pemrograman Web I dengan membuat website profil usaha minimal 4 halaman HTML (tercapai 7 halaman).
2. Mengimplementasikan konsep pemrograman web (HTML, CSS, JavaScript) secara praktis.
3. Melakukan modifikasi kreatif secara manual pada warna, font, layout, dan animasi.
4. Meng-hosting website secara online agar bisa diakses publik.
Manfaat:
1. Meningkatkan pemahaman mahasiswa tentang pengembangan website modern.
2. Memberikan pengalaman praktis membangun aplikasi web full-stack.
Bab II Pembahasan
2.1 Deskripsi Website
ZeeStore adalah sebuah website marketplace/toko online yang terinspirasi dari platform e-commerce besar seperti Tokopedia dan Shopee. Website ini merupakan profil usaha UMKM yang menyediakan berbagai produk dari kategori Elektronik, Fashion Pria, Fashion Wanita, Makanan & Minuman, Rumah Tangga, dan Olahraga.
Website dibangun sepenuhnya menggunakan HTML5, CSS3 (Bootstrap 5.3), dan JavaScript vanilla tanpa framework frontend, dengan backend menggunakan Supabase yang menyediakan PostgreSQL database dan sistem Authentication. Website terdiri dari 7 halaman HTML yang saling terintegrasi, 1 file CSS custom (style.css) berisi 1.097 baris kode styling, dan 8 file JavaScript modular untuk logika aplikasi.

2.3 Daftar Halaman Website
Website ZeeStore memiliki 7 halaman HTML (melebihi syarat minimal 4 halaman):
1. index.html - Halaman Login & Register. Mendukung login via email/password dan Google OAuth.
2. home.html - Halaman Utama / Beranda. Menampilkan hero banner promosi, scroll kategori, dan grid katalog.
3. product.html - Halaman Detail Produk. Menampilkan harga, deskripsi, stok, jumlah terjual, rating.
4. cart.html - Halaman Keranjang Belanja. Mengelola item di keranjang belanja, mengubah jumlah.
5. checkout.html - Halaman Checkout. Proses pembayaran multi-step.
6. sell.html - Halaman Jual Barang. Formulir untuk mempublikasikan produk baru.
7. profile.html - Halaman Profil Saya. Mengelola akun pengguna dan menampilkan riwayat belanja.

2.4 Teknologi yang Digunakan
Berikut adalah teknologi yang digunakan dalam pengembangan website ZeeStore:
1. HTML5 Semantik - Struktur halaman web menggunakan elemen semantik seperti <nav>, <main>, <section>, dan <footer>.
2. CSS3 + Bootstrap 5.3.3 - Styling custom sebanyak 1.097 baris ditambah framework responsif Bootstrap untuk grid system dan komponen UI.

2.7 Modifikasi Fungsionalitas
1. Perbaikan Bug Stok: Stok produk kini berkurang dan jumlah terjual (sold) bertambah setelah checkout. Menggunakan fungsi validasi database untuk mencegah stok negatif.
2. Penanganan JWT Expired: Otomatis logout dan redirect ke login jika token sudah expired, tanpa menampilkan error teknis ke user.
3. Sinkronisasi Alamat: Data alamat dari profil otomatis terisi di form checkout.
4. Keamanan XSS: Fungsi escapeHTML() dan sanitizeURL() untuk mencegah injection script.

Bab III Penutup
3.1 Kesimpulan
Website ZeeStore telah berhasil dikembangkan sebagai platform marketplace UMKM modern dengan 7 halaman HTML yang saling terintegrasi. Website dilengkapi dengan fitur autentikasi, katalog produk, keranjang belanja, proses checkout, jual barang, dan profil pengguna.
Modifikasi kreatif yang telah dilakukan meliputi perubahan tipografi (Outfit + Plus Jakarta Sans), sistem palet warna gradien dengan 28+ variabel CSS, penambahan efek glassmorphism, sistem shadow berlevel, micro-animations, serta penambahan halaman profil. Perbaikan logika seperti sinkronisasi otomatis antar form juga berjalan dengan baik.
3.2 Saran
Untuk pengembangan lebih lanjut, kami menyarankan penambahan fitur pencarian/filter lanjutan menggunakan query params URL, integrasi payment gateway (Midtrans/Xendit) untuk otomatisasi mutasi, fitur chat WebSocket antara pembeli dan penjual, dan sistem notifikasi pesanan real-time.
